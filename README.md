# Практические задачи по программированию c тестами

Для работы нужно установить [Node.js](https://nodejs.org/), рекомендуемые версии не ниже:
```
$ node -v
v14.18.2

$ npm -v
8.5.4
```

### Запуск

Установить зависимости
```
$ npm install
```

Прогнать тесты
```
$ npm run test
```

Проверить код на ошибки
```
$ npm run lint
```
